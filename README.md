#Snips APP to talk with your TV
==============================

A Snips Skill to control any remote controller on your local network.

Hardware Requirements:
---------------------
-A Raspberry Pi ~40$

-A mic ~25$

-A Speaker~10$

-A 4 GB Micro SD card~14$

-Broadlink RM3 Mini ~ 24$
